# Next-GraphQL-Cliente

> Práctica de desarrollo con Next.js y GraphQL

## Descripción
Este proyecto es una aplicación de gestión de clientes y pedidos, desarrollada como práctica para reforzar conocimientos en Next.js y GraphQL. Permite el registro de clientes, control de ventas y administración de productos y pedidos.

## Características principales
- Registro y autenticación de usuarios (JWT)
- Gestión de clientes y productos
- Creación y control de pedidos
- Validación de formularios con Formik y Yup
- Gestión de estado global con useReducer y Context API
- Interfaz moderna con Tailwind CSS
- Backend con Apollo Server y MongoDB (Mongoose)

## Tecnologías utilizadas
- **Next.js**
- **React.js**
- **Apollo Client & Apollo Server**
- **GraphQL**
- **Formik & Yup**
- **Tailwind CSS**
- **MongoDB & Mongoose**
- **JWT**

## Instalación y uso

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tuusuario/Next-GraphQL-Cliente.git
   ```
2. Instala las dependencias en ambos proyectos (`cliente` y `GraphQl`):
   ```bash
   cd cliente
   npm install
   cd ../GraphQl
   npm install
   ```
3. Configura las variables de entorno según corresponda.
4. Inicia el backend:
   ```bash
   cd GraphQl
   npm run dev
   ```
5. Inicia el frontend:
   ```bash
   cd ../cliente
   npm run dev
   ```
6. Accede a la app en [http://localhost:3000](http://localhost:3000)
